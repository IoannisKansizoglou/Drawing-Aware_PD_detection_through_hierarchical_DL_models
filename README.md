# Drawing-Aware PD detection

### Introduction

This repository contains an open-source toolbox for Parkinson's Disease (PD) detection from handwriting records based on PyTorch.

The main branch works with **PyTorch 1.8+**.

### What's New

* Release **Drawing-Type Classifier** (code and [weights](https://duth-my.sharepoint.com/:f:/g/personal/ikansizo_duth_gr/EohrhPpey5ZPhvJEhRi4wggBQRegu0VzsakZZ-SRll6MFA?e=uKOR5N)): a ResNet-18 model for classifying different drawing types (see drawing types below).
